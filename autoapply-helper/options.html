How to use
Quick start for the extension.
  • Fill your profile, URLs, snippets, and resume links in Settings, then click “Save profile”.
  • (Optional) Add job URLs to the Job queue. Enable auto-submit only if you trust the site.
  • On a job page, open the popup → “Prefill this page” (or press Ctrl+Shift+P palette on the page).
  • The tracker auto-logs the application. Open “Job Tracker” to edit statuses/notes.
  • Use “Get suggestions” to pull live roles, then “Add to queue” and run Mass apply from the popup.
  • For tailored answers, paste a JD + target role/company, then “Generate tailored answers”.
  • Chat with Grok in Settings if you need wording or setup help.
