Quick start guide for JobSeeker Pro+

How to Use JobSeeker Pro+

Follow these steps to get the most out of the extension

  1. Fill your profile, URLs, snippets, and resume links in Settings, then click "Save profile".
  2. (Optional) Add job URLs to the Job queue. Enable auto-submit only if you trust the site.
  3. On a job page, open the popup → "Prefill this page" (or press Ctrl+Shift+P palette on the page).
  4. The tracker auto-logs the application. Open "Job Tracker" to edit statuses/notes.
  5. Use "Get suggestions" to pull live roles, then "Add to queue" and run Mass apply from the popup.
  6. For tailored answers, paste a JD + target role/company, then "Generate tailored answers".
  7. Chat with Grok in Settings if you need wording or setup help.
